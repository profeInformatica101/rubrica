Estructuras de Datos en Python

Listas

Las listas en Python son colecciones ordenadas y mutables. Ejemplo:

mi_lista = [1, 2, 3, "cuatro", 5.0]

Diccionarios

Los diccionarios son colecciones no ordenadas que almacenan pares clave-valor. Ejemplo:

mi_diccionario = {"nombre": "Juan", "edad": 30, "ciudad": "Madrid"}

Conjuntos

Los conjuntos son colecciones no ordenadas de elementos únicos. Ejemplo:

mi_conjunto = {1, 2, 3, 3, 4}

Lista de Funciones Útiles en Python

  • len(): Devuelve la longitud de una colección.
  • type(): Devuelve el tipo de una variable.
  • print(): Imprime datos en la consola.
  • range(): Genera una secuencia de números.
