Práctica: Uso de ProtonVPN para Navegación Segura

Objetivos de la Práctica

  • Conocer qué es una VPN, su funcionamiento y usos principales.
  • Registrar, instalar y configurar ProtonVPN en modo gratuito.
  • Verificar el correcto funcionamiento de una VPN.
  • Reflexionar sobre la importancia de las VPN en el entorno digital.

Instrucciones

1. Registro en ProtonVPN

  1. Accede al sitio web oficial de ProtonVPN.
  2. Crea una cuenta gratuita completando el formulario de registro.
  3. Guarda tus credenciales de acceso (usuario y contraseña).

2. Instalación de ProtonVPN

Sigue los pasos según tu sistema operativo:

  • En Windows o macOS:
    1. Descarga el instalador oficial desde el sitio web de ProtonVPN.
    2. Ejecuta el instalador y sigue las instrucciones para completar la instalación.
  • En Linux:
    1. Abre una terminal y sigue las instrucciones oficiales para tu distribución. Ejemplo para distribuciones basadas en Debian/Ubuntu:
    2. sudo apt update
      sudo apt install -y openvpn
    3. Sigue la documentación oficial de Linux para instalar su cliente oficial.

3. Configuración y Conexión

  1. Inicia la aplicación ProtonVPN.
  2. Inicia sesión con tus credenciales.
  3. Conéctate a uno de los servidores gratuitos disponibles.
  4. Comprueba visualmente en la aplicación que la conexión esté activa.

4. Verificación del Funcionamiento

  1. Abre una terminal o símbolo del sistema en tu ordenador.
  2. Escribe y ejecuta el siguiente comando:
    3. curl ipinfo.io
  3. Observa los resultados y verifica que:
    • La dirección IP mostrada sea diferente a la de tu red local.
    • La ubicación corresponda al servidor de la VPN.
  4. Toma una captura de pantalla del resultado del comando.

5. Investigación: ¿Qué es una VPN?

Realiza una breve investigación y responde a las siguientes preguntas:

  1. ¿Qué es una VPN y cuál es su función principal?
  2. ¿Qué ventajas y desventajas tiene el uso de una VPN?
  3. ¿Qué datos cambian al usar una VPN y por qué esto es importante?

Escribe tus respuestas en el mismo documento que incluirá las capturas de pantalla.
