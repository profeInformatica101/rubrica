Práctica con OpenVAS

Guía para realizar la práctica paso a paso y explorar casos reales de uso

1. ¿Qué es OpenVAS y para qué sirve?

OpenVAS es una solución de código abierto utilizada para realizar análisis de vulnerabilidades en sistemas y redes. Es útil para:

  • Identificar puertos abiertos y servicios expuestos.
  • Detectar configuraciones inseguras en sistemas operativos y aplicaciones.
  • Generar informes sobre riesgos de seguridad y mitigarlos con soluciones específicas.
2. Instalación de OpenVAS

Linux: Abre una terminal y ejecuta los siguientes comandos:

sudo apt update
sudo apt install openvas -y

Una vez instalado, inicializa OpenVAS:

sudo gvm-setup
sudo gvm-check-setup

Windows/macOS: Usa una máquina virtual Linux

3. Configuración inicial

Accede al panel de administración web:

  1. Revisa el puerto en el que OpenVAS está escuchando:
    2. sudo gvm-start
  2. Abre tu navegador e ingresa la URL: https://<tu-ip>:9392.
  3. Usa las credenciales generadas durante la instalación (por defecto: admin).
4. Actualización de la base de datos de vulnerabilidades

Ejecuta el siguiente comando para actualizar las definiciones de vulnerabilidades:

sudo greenbone-feed-sync

Si ocurre algún error, asegúrate de que tu conexión a Internet esté activa.

5. Escaneo básico de vulnerabilidades

Realiza un escaneo de un sistema o red:

  1. En el panel, selecciona la opción Tasks.
  2. Crea un nuevo escaneo seleccionando un rango de IP o una sola IP.
  3. Inicia el análisis y espera a que finalice.
  4. Descarga el informe generado en formato PDF o HTML.
6. Práctica: Analiza un equipo de la red local

Escanea un equipo de tu red local (por ejemplo, tu máquina o un servidor de prueba). Sigue los pasos del escaneo básico y realiza un informe que contenga:

  • Los 3 riesgos más críticos detectados.
  • Las soluciones recomendadas para mitigarlos.
7. Automatización con tareas programadas

Configura un escaneo semanal para un segmento de red:

crontab -e

Agrega la siguiente línea:

0 2 * * 1 openvas-scan --target="192.168.1.0/24" >> ~/openvas_scan.log

Esto ejecutará el escaneo cada lunes a las 2:00 AM y guardará los resultados en openvas_scan.log.
