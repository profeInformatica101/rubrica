Práctica con ClamAV

Guía para realizar la práctica paso a paso

1. Instalación de ClamAV

Linux: Abre una terminal y ejecuta los siguientes comandos:

sudo apt update
sudo apt install clamav clamav-daemon -y

Windows/macOS: Descarga ClamAV desde su sitio oficial y sigue las instrucciones.

2. Actualización de la base de datos de virus

Ejecuta el siguiente comando para actualizar las definiciones de virus:

sudo freshclam
Si ocurre error
3. Descarga del archivo de prueba EICAR

Visita el enlace: EICAR Test File.

Descarga el archivo eicar.com y guárdalo en tu sistema.

4. Configuración para la práctica

Crea un directorio y copia el archivo de prueba:

mkdir ~/clamav_test
mv ~/Descargas/eicar.com ~/clamav_test/
5. Escaneo con ClamAV

Realiza un escaneo del directorio:

clamscan -r ~/clamav_test/

Resultado esperado:

ClamAV detectará el archivo EICAR como una amenaza.

6. Limpieza y eliminación de amenazas

Para eliminar el archivo detectado automáticamente:

clamscan --remove=yes -r ~/clamav_test/
