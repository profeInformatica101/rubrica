Malware y Seguridad Informática

Introducción al Malware

El objetivo es comprender cómo funciona el software malicioso o malware, que agrupa virus, gusanos, troyanos y, en general, todos los programas diseñados para acceder a los ordenadores sin autorización y causar efectos no deseados.

  • En los comienzos, la motivación principal de los creadores de virus era el reconocimiento público.
  • Con la evolución de las tecnologías de comunicación y su implantación en todos los aspectos de la vida diaria, los ciberdelincuentes han visto en ello un negocio muy lucrativo y han pasado a tener una motivación económica.

*Formas de obtener un beneficio económico por parte del creador de un programa malicioso*

  • Robar información sensible del ordenador infectado, como contraseñas, correos electrónicos, datos personales y credenciales de acceso a diferentes entidades, incluyendo banca online.
  • Crear una red de ordenadores infectados llamados Botnet o red Zombi. Esto permite al atacante manipularlos simultáneamente para realizar ataques de denegación de servicio (DDoS) o enviar spam masivo.
  • Venta de falsas soluciones de seguridad (rogueware) que no realizan lo que afirman, como los falsos antivirus.
  • Cifrar el contenido de los ficheros del ordenador y solicitar un rescate económico al usuario para recuperar la información (ransomware).

Motivaciones

  • Reconocimiento público (históricamente).
  • Motivaciones económicas debido a la evolución tecnológica.

Formas de obtener beneficios económicos

  • Robo de información sensible como contraseñas, correos, y datos bancarios.
  • Creación de redes de ordenadores infectados (Botnet).
  • Venta de soluciones de seguridad falsas (Rogueware).
  • Cifrado de archivos y solicitud de rescates económicos.

Clasificación del Malware

Tipo Descripción
Virus Infectan archivos y dependen de ellos para ejecutarse.
Gusano Se replica y propaga automáticamente a través de redes y dispositivos.
Troyano Crea una puerta trasera para accesos no autorizados.
Infostealers Programas maliciosos que roban información.
Crimeware Software diseñado para cometer delitos.
Greyware Aplicaciones molestas pero no dañinas.

Métodos de Infección

  • Explotación de vulnerabilidades en sistemas y software.
  • Ingeniería social para engañar a los usuarios.
  • Archivos maliciosos en correos o descargas.
  • Dispositivos extraíbles infectados.
  • Cookies maliciosas que registran actividades en Internet.

Protección y Desinfección

  • Mantener el equipo actualizado y usar software legal.
  • Hacer copias de seguridad frecuentes.
  • Usar contraseñas fuertes y cuentas con pocos privilegios.
  • Utilizar herramientas antimalware confiables.

Pasos para limpieza de malware

  1. Desconectar de la red.
  2. Identificar procesos y drivers maliciosos.
  3. Finalizar los procesos maliciosos.
  4. Eliminar archivos maliciosos.
  5. Reiniciar el equipo y repetir.

Clasificación de Técnicas de Ataque

  • Malware: Programas maliciosos como virus, gusanos, troyanos, etc.
  • Phishing: Suplantación de identidad mediante correos electrónicos, mensajes falsos o páginas web clonadas.
  • Spear Phishing: Variante de phishing dirigida a individuos específicos, como empleados de una empresa.
  • Whaling: Ataques dirigidos a altos ejecutivos o figuras clave de una organización.
  • Scam: Estafas electrónicas para obtener información o dinero mediante engaños.
  • Hoax: Bulos o engaños diseñados para generar pánico o confusión.
  • Sniffing: Interceptación de tráfico de red para capturar datos confidenciales.
  • Spoofing: Suplantación de identidad en correos, direcciones IP o dominios.
  • Man-in-the-Middle (MitM): Interceptación de comunicación entre dos partes para espiar, modificar o robar información.
  • SQL Injection: Inserción de código malicioso en consultas SQL para acceder a bases de datos.
  • Cross-Site Scripting (XSS): Inserción de scripts maliciosos en páginas web para afectar a los usuarios.
  • Cross-Site Request Forgery (CSRF): Forzar a un usuario autenticado a ejecutar acciones no deseadas.
  • Tabnabbing: Suplantación de pestañas del navegador para capturar credenciales.
  • Denegación de Servicio (DoS): Saturar un recurso o servicio para que sea inaccesible.
  • Denegación de Servicio Distribuida (DDoS): Variante de DoS que utiliza múltiples dispositivos infectados.
  • Brute Force: Intentos repetidos de adivinar contraseñas o claves de acceso.
  • Password Cracking: Descifrar contraseñas utilizando herramientas o bases de datos de contraseñas comunes.
  • Botnet: Red de dispositivos infectados controlados remotamente.
  • Ransomware: Cifrado de datos del usuario seguido de una demanda de rescate para descifrarlos.
  • Keylogging: Registro de las pulsaciones del teclado para capturar contraseñas y datos personales.
  • Drive-by Download: Descarga automática de malware al visitar una página web comprometida.
  • Watering Hole Attack: Compromiso de sitios web frecuentados por un objetivo específico para infectar a los visitantes.
  • Social Engineering: Manipulación psicológica para obtener información o acceso al sistema.
  • Credential Stuffing: Uso de credenciales filtradas en diferentes servicios para acceso no autorizado.
  • Zero-Day Exploit: Ataques que aprovechan vulnerabilidades desconocidas para los desarrolladores.
  • Fileless Malware: Malware que opera en la memoria del sistema sin escribir archivos en el disco.
  • Session Hijacking: Robo de una sesión activa para obtener acceso no autorizado a una aplicación o servicio.
  • DNS Spoofing: Redirección de usuarios a sitios web maliciosos falsificando respuestas DNS.
  • Cryptojacking: Uso no autorizado de recursos del sistema para minar criptomonedas.
  • Watermarking Malware: Marcado de documentos o archivos con identificadores maliciosos.
  • Ad Fraud: Generación de clics falsos en anuncios para obtener ingresos ilícitos.
  • Exploit Kits: Herramientas automatizadas para buscar y explotar vulnerabilidades en sistemas desactualizados.
  • Backdoors: Creación de accesos ocultos para el atacante.
  • Remote Access Trojans (RATs): Herramientas que permiten el control remoto completo del dispositivo infectado.
