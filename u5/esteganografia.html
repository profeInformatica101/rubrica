Esteganografía

¿Qué es la Esteganografía?

La esteganografía es la técnica de ocultar información dentro de archivos multimedia como imágenes, videos o audios. A diferencia de la criptografía, el objetivo es ocultar la existencia misma del mensaje.

¿Cómo funciona?

La esteganografía modifica un archivo portador, como una imagen, para ocultar datos en su interior sin alterar su apariencia o funcionalidad. Esto se combina frecuentemente con criptografía para aumentar la seguridad.

Aplicaciones:

  • Protección de datos sensibles.
  • Marcado digital para derechos de autor.
  • Comunicaciones seguras y privadas.

Ejemplo Práctico en Linux: Enviar Información Confidencial

Contexto del Ejemplo:

Imagina que necesitas enviar un archivo con credenciales a un compañero de trabajo sin que se note. En este caso, ocultaremos un archivo de texto dentro de una imagen usando steghide.

Paso 1: Preparar el Entorno

Instalar steghide: Si no lo tienes instalado, ejecuta:

sudo apt update
sudo apt install steghide

Crear los archivos necesarios:

Archivo con la información confidencial:

echo "Usuario: admin" > credenciales.txt
echo "Contraseña: password123" >> credenciales.txt

Archivo portador: Usa una imagen, por ejemplo, imagen.jpg.

Paso 2: Ocultar el Archivo

Utiliza steghide para incrustar el archivo credenciales.txt dentro de la imagen imagen.jpg:

steghide embed -cf imagen.jpg -ef credenciales.txt

Te pedirá establecer una contraseña para proteger el proceso de esteganografía. Introduce una contraseña segura.

Paso 3: Verificar el Archivo Oculto

El archivo portador (imagen) no tendrá cambios visibles, pero el tamaño del archivo puede aumentar ligeramente. Puedes verificar los metadatos con: 

file imagen.jpg

Paso 4: Extraer el Archivo

Cuando el destinatario reciba el archivo portador, puede extraer el contenido oculto con el siguiente comando: 

steghide extract -sf imagen.jpg

Te pedirá la contraseña que definiste durante el proceso de ocultación. Una vez ingresada, el archivo credenciales.txt será extraído.

Resultado

  • Antes de ocultar: La imagen era simplemente un archivo visual.
  • Después de ocultar: La imagen contiene información confidencial que puede ser extraída con la contraseña correcta.

Ventajas del Ejemplo

  • Bajo perfil: La imagen parece inofensiva, lo que evita levantar sospechas.
  • Seguridad adicional: Incluso si alguien encuentra el archivo portador, necesitará la contraseña para extraer los datos.
  • Compatibilidad: Puedes usar este método con cualquier archivo multimedia.
